# Blog application
### In this blog, people can exchange different events with each other

To run local:
1. Clone repo
```bash
git clone https://github.com/WIUT00014541/WebTech_CW2_00014541.git
```
2. Install libs
```bash

npm install
```
3. Run the app
```bash
node app.js
```

### Libs
- express.js
- pug.js
